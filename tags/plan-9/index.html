Posted: 2020-08-15   |   Tags: Plan 9

This document outlines how to set up Plan 9 (9front to be exact) to be used with drawterm. Readers are not expected to have any Plan 9 experience. Basic Unix knowledge is helpful but not a requirement. At the end of this tutorial you will have the following: a virtual machine (VM) runnning 9front the ability to remotely connect to the VM using drawterm This tutorial has been tested on Windows 10 and VirtualBox.
